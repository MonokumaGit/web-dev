The path to enlightenment

these instructions are for *nix platforms. We also have Windows instructions.

You can run the tests by calling the path to enlightenment.rb file

In your terminal, while in the ruby koans directory, type

[ ruby_koans ] 1 ruby path_to_enlightenment.rb

Red, green, refactor

In test-driven development (tdd) the mantra has always been red: write a
failing test and run it, green: make the test pass, and refactor: look at the code
and see if you can make it any better
