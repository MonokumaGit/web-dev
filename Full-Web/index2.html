Главная

Функция памяти

  1. Прием информации от других устройств
  2. Хранения информации
  3. Передача информации по запросу в другие устройства комьютера

Память делят на:

Основную(или внутреннеюю)

  • Озу (оператвиное запомниющиее устройство),по-английски RAM (Random Access Memory - "память с произвольным доступом")
  • ПЗУ(постоянное запомниющиее устройство), по-английски ROM (Read only Memory - "память только для чтения")

Внешнюю(устройство внешней памяти позволяет длительнно хранить информацию)

  • Носитель внешней памяти:жесткие и гибкие магнитные диски, а также лазерные диски и флэш-карты
